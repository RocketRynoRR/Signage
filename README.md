# GitHub Pages Projects

This repository contains GitHub Pages webpages and mini-apps.

## Digital Ad Board

The digital advert board app is in:

```text
digital-ad-board/
```

Main pages:

```text
digital-ad-board/display.html
digital-ad-board/admin.html
```

- `display.html` is the full-screen TV slideshow.
- `admin.html` is the Supabase-powered admin page for slides, logos, captions, overlays, and board colours.

Supabase setup:

```text
digital-ad-board/supabase/schema.sql
```

Run that SQL file in Supabase SQL Editor whenever the database structure needs updating.

More detailed setup notes are here:

```text
digital-ad-board/README.md
```

## GitHub Pages

The root `index.html` is a simple project menu. Once GitHub Pages is enabled, the digital ad board pages will be available under the `digital-ad-board` folder path for this repository.
